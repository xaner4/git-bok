På Linux så er som oftest Git forhands installert,
Du kan sjekke med å kjøre 

```Bash
$ git --version
```

#### Debian / Ubuntu

```bash
$ apt update
$ apt install git -y
```

#### RedHat/Fedora

```bash
$ yum install git-core
```

Eller 


```bash
$ dnf install git-core
```



#### Arch Linux

```bash
$ pacman -S git
```
