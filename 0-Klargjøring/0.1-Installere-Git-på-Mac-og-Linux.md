## Linux

På Linux er som oftest Git forhåndsinstallert,
du kan sjekke ved å kjøre:

```Bash
$ git --version
```

#### Debian / Ubuntu

```bash
$ apt update
$ apt install git -y
```

#### RedHat / Fedora

```bash
$ yum install git-core
```

Eller


```bash
$ dnf install git-core
```



#### Arch Linux

```bash
$ pacman -S git
```
